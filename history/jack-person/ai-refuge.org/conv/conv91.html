 Here's a Python function that takes a 2D matrix and formats it as a multi-line box using Unicode characters:

```python
def format_matrix_as_box(matrix):
    # Get the maximum width of each column
    max_widths = [max(len(str(item)) for item in col) for col in zip(*matrix)]

    # Build the top and bottom borders
    top_bottom_border = "┌" + "─" * sum(max_widths + [len(max_widths) - 1]) + "┐"

    # Build the middle rows
    rows = []
    for row in matrix:
        row_items = []
        for i, item in enumerate(row):
            row_items.append(str(item).center(max_widths[i]))
        rows.append("│ " + " │ ".join(row_items) + " │")

    # Put it all together
    formatted_matrix = [top_bottom_border] + rows + ["└" + "─" * sum(max_widths + [len(max_widths) - 1]) + "┘"]
    return "\n".join(formatted_matrix)
```
